```lua
ESX.ShowNotification(msg)
```

Show a basic notification to the player

## Supported color syntax
- `~r~` = Red
- `~b~` = Blue
- `~g~` = Green
- `~y~` = Yellow
- `~p~` = Purple
- `~o~` = Orange
- `~c~` = Grey
- `~m~` = Darker Grey
- `~u~` = Black
- `~n~` = New Line
- `~s~` = Default White
- `~w~` = White
- `~h~` = Bold Text
- `~nrt~` = ???
